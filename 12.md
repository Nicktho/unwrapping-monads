# Example - Identity

Alternatively, it might be cleaner to create a class `Identity`

```javascript

class Identity {
  contructor(value) {
    this._value = value
  }

  bind(fn) {
    return fn(this._value)
  }
}

const monad = new Identity(1)

monad
  .bind(a => new Identity(a + 1))
  .bind(a => new Identity(a * 2))
  .bind(a => assert(a, 4))

```
